# Sales Analytics Dashboard

## Overview
Built an end-to-end sales analytics project to generate business insights from customer, product, and transaction data.

## Tools Used
Python (Pandas), Power BI, Matplotlib

## Workflow
Data Cleaning → EDA → KPI Creation → Dataset Merge → Dashboard

## Key Features
- Revenue & KPI tracking
- Monthly sales trend
- Customer & product insights
- Payment & shipping analysis

## Insights
- Electronics drives highest revenue
- Credit Card is most used payment method
- Majority ratings are positive (4–5)

## Dashboard
Interactive Power BI dashboard for business decision-making
